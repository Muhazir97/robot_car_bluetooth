================= ROBOT CAR BLUETOOTH + LED ====================

===== ARDUINO TO DRIVER =====
PIN 2 => IN 1
PIN 3 => IN 2
PIN 4 => IN 3
PIN 5 => IN 4

===== BLUETOOTH TO ARDUINO =====
RX  => TX
TX  => RX
GND => GND
VCC => 5V

===== ARDUINO TO LED =====
PIN A0 => LED DEPAN KANAN
PIN A1 => LED DEPAN KIRI
PIN A2 => LED BELAKANG KANAN
PIN A3 => LED BELAKANG KIRI
     ALL GND TO GND 

====== POWER ======
KIRI   = POSITIVE
TENGAH = NEGATIVE
